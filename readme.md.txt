## Requisitos

- Node.js 20.x
- Angular CLI 18.x
- .NET 8.0
- Visual Studio 2022
- XAMPP (para base de datos MySQL)

## Levantar Backend (UrbanIntelAPI)

1. Abrir `UrbanIntelAPI` en Visual Studio.
2. Restaurar paquetes NuGet. (revisar librerias en paquetes nuget si es la primera vez que se levanta el sistema)
3. Correr el proyecto (`Ctrl + F5`).

## Levantar Frontend (UrbanIntelWEB)

1. Entrar a la carpeta `UrbanIntelWEB/urbanintelweb`.
2. Correr `npm install`.
3. Correr `ng serve`.

## Notas
- Swagger disponible en http://localhost:5016/ (revisar las rutas)
- Angular disponible en http://localhost:4200/
